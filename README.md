# 🚀 Data Structures From Scratch (C++)

This project is part of my learning journey where I implemented core data structures from scratch instead of relying on built-in ones.

---

## 📌 Implemented Structures
- Dynamic Array
- Stack (using Array)
- Queue (using Array)

---

## 💡 Key Idea

The main goal of this project was not just writing code, but understanding how memory works behind the scenes.

Instead of using ready-made structures, I built everything manually to deeply understand:
- How data is stored in memory
- How data moves during insertions and deletions
- How resizing affects performance and structure behavior

---

## 🛠 Core Operations Implemented
- resize
- insert
- delete
- find
- reverse

---

## 🧠 What I Learned

- Data Structures are not just concepts to memorize
- They are ways to organize and control data efficiently
- Stack (LIFO) is based on last-in-first-out behavior
- Queue (FIFO) is based on first-in-first-out behavior
- Understanding memory makes problem solving much clearer and more logical
